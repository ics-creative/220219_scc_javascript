# Vue 3 + Vite

このテンプレートは、ViteでVue 3を使った開発を始めるのに役立つはずです。テンプレートはVue 3の `<script setup>` SFCを使用しています。詳しくは [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) をチェックしてください。
